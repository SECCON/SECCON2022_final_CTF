HowToPlay

Summary

This problem is designed to give you a lot of questions, and the content is to answer as many of them correctly as possible. Some problems have over 100,000 questions.

In principle, questions are done by comparing arrays, and one point is awarded for a match between the array index and the value. For example, if the correct ans The scoring instructions for the questions can be downloaded as source code from each question page. for example, the answer is [1,2,3,4,5] and you submit [1,1,1,1,5], you will receive 2 points. However, there are exceptions. There are cases where you get 0 points if you make a mistake in a specific number, and there are also problems where points are gained after a certain amount of time has passed. See the source code for each issue for details.

The scoring instructions for the questions can be downloaded as source code from each question page.

main points of this page
  1. You can only submit once in 1 tick (5 seconds)
  2. There is a problem with different answers every tick
  3. Some requests may not return for several seconds. Despite our best efforts, there is no compensation for missed ticks due to unreturned requests. ((DO NOT DoS))
  4. If you miss a tick, you can specify a past tick and submit it. (There is a 5 second submission limit)
  5. For each stage, KoH score will be calculated by the highest score per 5 minutes
  6. KoH scores are determined only by rank. Rank is determined solely by the number of points earned in a stage.

What's Stage?

There will be a total of 4 questions. The problems are called STAGE and range STAGE1~STAGE4. Each question will not be posed in parallel at the same time.

How are KoH scores calculated? / What's Round?

Every 5 minutes, the score is calculated based on the highest score at that point in time at the stage being held . This unit of progress every 5 minutes is round . In other words, when stage2 is being held, the score is calculated every 5 minutes based on the highest score at that time in stage2.

The Koh score differs from the question score (number of correct answers). KoH scores are determined by the rank order of scores per 5 minutes. The rankings and KoH points are as follows. Even if you score 100,000 points, if you are in first place, you get 20 points for that round.

What's tick?

You can only answer a question once every 5 seconds. This 5-second unit is the tick . Thus, one round is 60 ticks.

There is a problem where the correct answer changes with each tick.

For example, [1,2,3,4,5] in tick 1 of stage 1 may become [2,3,4,5,6] in tick 2 of stage1.

How it changes at each tick is specified in the source code distributed at each stage.

Each tick can be submitted once , and can also be submitted as a past tick by specifying it in the request.

How To Submit?

You can submit the following source code.

note: You cannot check the content of the submission, so please manage it yourself. 

import requests
import json
url = "http:///api/quiz/"

your_token = # please enter your token
headers = {'Authorization': 'Token ' + your_token}

params = json.dumps({
  'answer': [0,1,2,3,4], # [required] your answer (Substituted into `your_answer` in the code)
  'stage': 1, # [required] this stage number. please use this value for this stage :)
  'tick': 180 # [optional] specify the current tick or less (If omitted, current tick is set)
  })
res = requests.post(url, data=params, headers=headers)

# Returns 200 if accepted. otherwise returns not 200
if res.status_code == 200:
  print(res.text)
else:
  print(res.text)

PROHIBITIONS

This KoH assumes that you will score points solely by answering questions. Please do not attack the scoring server by exploiting web vulnerabilities. Also, questions are only accepted at 5 second intervals. Please do not send requests that will overload the server, such as DoS.
